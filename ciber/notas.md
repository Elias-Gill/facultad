MOdelos de control de acceso

Un modelo es un conjunto de criterios para determinar quien puede ingresar al sistema
- Control de acceso obligatorio
- Control de acceso discrecioneal
- Control de acceso basado en roles

# Para el Tp

- Fase 1: definir el alcance del proyecto
- Fase 2: identificar nuestros activos (las partes mas importantes del departamenteo, ejemplo servidores, routers, etc)
- Fase 3: identificar las *posibles amenazas* que pueden recibir cada activo
- Fase 4: identificar las *vulnerabilidades* que sufren nuestros activos
- Fase 5: evaluar el riesgo que implican dichas vulnerabilidades de nuestros activos. Pueden haber vulnerabilidades que no
          requieran realizar acciones innmediatas
- Fase 6: realizar las acciones de correcion de las vulnerabilidades.
